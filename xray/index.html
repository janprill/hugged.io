X-ray

A chrome plugin for aligning objects to a grid.

Align elements on an 8 or 16px grid.

Visualize how elements align to a background grid. It's similar to using background grids in illustrator, photoshop, and sketch. This chrome extensions allows you to toggle the outline on elements and switch between an 8 and 16px grid in the background.

Get the Extension
